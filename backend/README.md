# CRA MiniDesk Backend

This directory will host the backend service for CRA MiniDesk.

## 🛠️ Planned Stack

*   **Runtime**: Java 21 (LTS)
*   **Framework**: Spring Boot (REST API architecture)
*   **Database**: PostgreSQL
*   **Migrations**: Flyway
*   **Build Tool**: Maven with Maven Wrapper (`mvnw`)

## 📌 Implementation Notes

The backend is scheduled to be scaffolded during Sprint 1. It will expose a clean, secure RESTful API to communicate with the frontend/desktop agent. No swagger/OpenAPI tools will be used, keeping API interactions straightforward, type-safe, and secure.
